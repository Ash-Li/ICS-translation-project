## 字幕格式

英/中字幕均采用`SRT`格式。SRT（Subripper）是最简单的文本字幕格式，扩展名为.srt，其组成为：一行字幕序号，一行时间代码，一行字幕数据。

```
45
00:02:52,184 --> 00:02:53,617
慢慢来
```
这表示：第45个字幕，显示时间从该视频开始的第2分52.184秒到第2分53.617秒，内容为：慢慢来。

> 摘自维基百科

## 推荐工具

`.SRT`为纯文本字幕格式，任何**文本编辑器**都可以轻松编辑。
